A chatbot built using the Gemini API, similar to ChatGPT, deployed on Render. This project leverages modern web technologies to provide a seamless chat experience.

## Tech Stack

![Express](https://img.shields.io/badge/Express-4B3C29?style=for-the-badge&logo=express&logoColor=white) - **Express**: A fast web framework for Node.js, perfect for building APIs.

![Axios](https://img.shields.io/badge/Axios-5A29E4?style=for-the-badge&logo=axios&logoColor=white) - **Axios**: A promise-based HTTP client for the browser and Node.js, used for making requests to the Gemini API.

![Vite](https://img.shields.io/badge/Vite-4A4A4A?style=for-the-badge&logo=vite&logoColor=white) - **Vite**: A modern frontend build tool that provides a fast development environment.

![Framer Motion](https://img.shields.io/badge/Framer%20Motion-00D8FF?style=for-the-badge&logo=framer&logoColor=white) - **Framer Motion**: A library for animations in React applications, enhancing the user experience with smooth transitions.

![Gemini API](https://img.shields.io/badge/Gemini%20API-FF5733?style=for-the-badge&logo=api&logoColor=white) - **Gemini API**: A powerful interface for integrating AI capabilities into applications, enabling developers to build intelligent chatbots and enhance user interactions seamlessly.

![Redux Toolkit](https://img.shields.io/badge/Redux%20Toolkit-764ABC?style=for-the-badge&logo=redux&logoColor=white) - **Redux Toolkit**: A standardized way to write Redux logic that simplifies the process of managing state.


## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ShardenduMishra22/AI-Chat-Bot.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Ai-chat-bot
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```
