# Dragon Ball API

## Introduction

Welcome to the documentation for the Dragon Ball API! This API is designed to provide access to various data related to the Dragon Ball universe, enabling developers to build exciting applications and features.

### Project Overview

The Dragon Ball API is built using TypeScript, Node.js and Express, with MongoDB for data storage and JWT for authentication. It aims to deliver a seamless experience for users looking to interact with Dragon Ball characters, episodes, and more.

### Features

- **Character Data**: Access detailed information about your favorite Dragon Ball characters.
- **Authentication**: Secure endpoints using JSON Web Tokens (JWT) for user authentication.
